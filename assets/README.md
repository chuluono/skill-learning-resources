# 视觉元素和图标

## 目录结构

```
assets/
├── icons/         # 项目图标
├── images/        # 项目图片
└── README.md      # 本说明文件
```

## 图标使用指南

### 图标类型

- **技能图标**：用于表示不同领域和技能的图标
- **导航图标**：用于网站导航和功能按钮的图标
- **状态图标**：用于表示不同状态（成功、警告、错误等）的图标

### 使用方法

1. **Markdown中使用**：
   ```markdown
   ![Python编程](../assets/icons/python.png)
   ```

2. **HTML中使用**：
   ```html
   <img src="../assets/icons/python.png" alt="Python编程" width="48" height="48">
   ```

## 图片使用指南

### 图片类型

- **技能封面**：用于技能页面的封面图片
- **示例图片**：用于展示教程和示例的图片
- **图表**：用于数据可视化和概念说明的图表

### 使用方法

1. **Markdown中使用**：
   ```markdown
   ![Python学习路径](../assets/images/python-learning-path.png)
   ```

2. **HTML中使用**：
   ```html
   <img src="../assets/images/python-learning-path.png" alt="Python学习路径" width="600" height="400">
   ```

## 图标和图片规范

### 尺寸

- **图标**：建议使用48x48像素或64x64像素
- **封面图片**：建议使用1200x600像素
- **示例图片**：建议使用800x600像素
- **图表**：根据内容需要调整尺寸

### 格式

- **图标**：优先使用PNG或SVG格式
- **图片**：优先使用PNG或JPEG格式
- **图表**：优先使用PNG或SVG格式

### 命名规范

- 使用小写字母和下划线
- 清晰描述图标或图片的内容
- 示例：`python_programming.png`、`ui_design_cover.jpg`

## 贡献指南

### 添加新图标

1. 确保图标符合项目的视觉风格
2. 按照命名规范命名图标文件
3. 将图标放入相应的目录
4. 更新本README文件（如有必要）

### 添加新图片

1. 确保图片质量高且相关
2. 按照命名规范命名图片文件
3. 将图片放入相应的目录
4. 更新本README文件（如有必要）

## 推荐工具

- **图标创建**：Figma、Adobe Illustrator、Inkscape
- **图片编辑**：Adobe Photoshop、GIMP、Canva
- **图表创建**：Microsoft Excel、Google Sheets、Tableau

## 注意事项

- 确保所有图标和图片都有适当的版权许可
- 优先使用开源或免费的图标和图片资源
- 对于商业用途的图标和图片，确保获得必要的授权
- 优化图片大小以提高加载速度

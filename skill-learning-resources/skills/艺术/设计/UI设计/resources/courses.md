# UI设计推荐课程

## 入门级课程

### 1. 网易云课堂 - 《UI设计基础入门》
- **平台**：网易云课堂
- **讲师**：Miu
- **课程时长**：约20小时
- **推荐理由**：国内知名UI设计师授课，内容全面易懂
- **内容亮点**：设计基础、软件操作、项目实战
- **适合人群**：UI设计零基础初学者
- **价格**：付费（约199元）

### 2. 优达学城 - 《用户体验设计入门》
- **平台**：优达学城 (Udacity)
- **讲师**：Google设计团队
- **课程时长**：约10周
- **推荐理由**：Google出品，质量有保障，理论与实践结合
- **内容亮点**：用户研究、原型设计、可用性测试
- **适合人群**：希望了解UX/UI设计的新手
- **价格**：免费（基础版）/ 付费（纳米学位）

### 3. Coursera - 《设计思维基础》
- **平台**：Coursera
- **讲师**：University of Virginia
- **课程时长**：约4周
- **推荐理由**：名校课程，系统性讲解设计思维方法
- **内容亮点**：同理心、定义问题、头脑风暴、原型设计
- **适合人群**：希望培养设计思维的初学者
- **价格**：免费（审计模式）/ 付费（证书）

### 4. B站 - 《Figma零基础入门到精通》
- **平台**：B站
- **讲师**：设计师小川
- **课程时长**：约15小时
- **推荐理由**：免费系统学习Figma，适合现代UI设计工作流
- **内容亮点**：Figma基础操作、组件库构建、原型设计
- **适合人群**：需要掌握Figma工具的设计师
- **价格**：免费

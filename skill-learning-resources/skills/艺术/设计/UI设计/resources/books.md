# UI设计推荐书籍

## 入门级书籍

### 1. 《设计心理学》系列
- **作者**：唐纳德·A·诺曼
- **出版社**：中信出版社
- **推荐理由**：从用户心理角度解释设计原则，帮助理解用户行为和需求
- **内容亮点**：可用性设计、用户体验、设计原则
- **适合人群**：所有UI设计初学者

### 2. 《写给大家看的设计书》
- **作者**：Robin Williams
- **出版社**：人民邮电出版社
- **推荐理由**：简单易懂的设计基础入门书，涵盖四大设计原则
- **内容亮点**：对比、重复、对齐、亲密性四大设计原则
- **适合人群**：设计基础薄弱的初学者

### 3. 《UI设计从零开始》
- **作者**：Miu
- **出版社**：清华大学出版社
- **推荐理由**：国内UI设计入门经典，结合实际案例讲解
- **内容亮点**：软件操作、设计流程、项目实战
- **适合人群**：希望快速上手UI设计的新手

### 4. 《Figma实用指南》
- **作者**：Daniel Schwarz
- **出版社**：人民邮电出版社
- **推荐理由**：Figma工具使用详解，适合现代UI设计工作流
- **内容亮点**：Figma操作技巧、组件库构建、团队协作
- **适合人群**：需要掌握Figma工具的设计师

## 中级书籍

### 1. 《用户体验要素》
- **作者**：Jesse James Garrett
- **出版社**：机械工业出版社
- **推荐理由**：用户体验设计的经典之作，提出五层模型
- **内容亮点**：战略层、范围层、结构层、框架层、表现层
- **适合人群**：希望深入理解UX设计的UI设计师

### 2. 《交互设计精髓》
- **作者**：Alan Cooper 等
- **出版社**：电子工业出版社
- **推荐理由**：交互设计领域的权威书籍，系统讲解交互设计原则
- **内容亮点**：目标导向设计、人物角色、场景设计
- **适合人群**：需要提升交互设计能力的UI设计师

### 3. 《设计系统》
- **作者**：Marco Suarez 等
- **出版社**：O'Reilly Media
- **推荐理由**：详细讲解设计系统的构建方法和实践
- **内容亮点**：组件库、设计规范、风格指南
- **适合人群**：需要构建设计系统的团队设计师

### 4. 《微动效设计》
- **作者**：Val Head
- **出版社**：人民邮电出版社
- **推荐理由**：专注于UI微动效设计，提升界面体验
- **内容亮点**：动画原理、微交互、动效实现
- **适合人群**：希望提升动效设计能力的UI设计师

## 高级书籍

### 1. 《设计思维》
- **作者**：Tim Brown
- **出版社**：中信出版社
- **推荐理由**：IDEO创始人讲解设计思维方法，提升创新能力
- **内容亮点**：同理心、定义问题、头脑风暴、原型、测试
- **适合人群**：希望提升设计思维和创新能力的设计师

### 2. 《精益设计》
- **作者**：Jeff Gothelf 等
- **出版社**：人民邮电出版社
- **推荐理由**：结合精益创业思想，优化设计流程
- **内容亮点**：假设验证、快速迭代、用户反馈
- **适合人群**：产品设计师和设计团队负责人

### 3. 《无障碍设计》
- **作者**：Sarah Horton 等
- **出版社**：O'Reilly Media
- **推荐理由**：详细讲解无障碍设计原则和实践
- **内容亮点**：WCAG标准、屏幕阅读器、键盘导航
- **适合人群**：需要设计无障碍产品的UI设计师

### 4. 《设计管理》
- **作者**：刘军
- **出版社**：北京大学出版社
- **推荐理由**：国内设计管理领域权威书籍，讲解设计团队管理
- **内容亮点**：设计团队建设、项目管理、设计流程优化
- **适合人群**：设计团队负责人和高级设计师

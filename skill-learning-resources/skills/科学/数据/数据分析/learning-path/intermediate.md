# 数据分析中级学习路径

## 阶段目标

- 掌握数据清洗和处理技术
- 学习统计分析方法和假设检验
- 精通高级数据可视化技术
- 掌握机器学习基础算法
- 应用数据分析解决特定领域问题

## 学习内容

### 1. 数据处理

#### 核心概念
- **数据清洗**：处理缺失值、异常值、重复值
- **数据转换**：标准化、归一化、编码分类变量
- **数据集成**：合并多个数据源
- **特征工程**：创建和选择有意义的特征
- **数据抽样**：处理大规模数据的抽样方法

#### Python实现
- **Pandas高级**：数据框操作、分组聚合、窗口函数
- **NumPy高级**：数组操作、向量化计算
- **数据处理库**：Dask（处理大规模数据）

#### 学习资源
- 书籍：《Python数据分析》、《Pandas实战》
- 课程：数据清洗与预处理、特征工程
- 文章：Pandas高级技巧、数据清洗最佳实践
- 工具：Pandas、NumPy、Dask

### 2. 统计分析

#### 描述性统计
- **集中趋势**：均值、中位数、众数
- **离散程度**：方差、标准差、四分位数
- **分布特征**：偏度、峰度

#### 推断性统计
- **参数估计**：点估计、区间估计
- **假设检验**：t检验、方差分析、卡方检验
- **相关分析**：皮尔逊相关、斯皮尔曼相关
- **回归分析**：线性回归、多元回归

#### 学习资源
- 书籍：《应用统计学》、《统计学习方法》
- 课程：统计分析方法、假设检验
- 文章：回归分析详解、假设检验步骤
- 工具：SciPy、StatsModels、R

### 3. 数据可视化高级

#### 交互式可视化
- **工具**：Plotly、Bokeh、Dash
- **交互元素**：缩放、悬停、筛选、联动
- **仪表板**：创建综合数据仪表板

#### 专业可视化工具
- **Tableau**：连接数据源、创建工作表、仪表板
- **Power BI**：数据导入、DAX函数、可视化设计
- **QlikView/Qlik Sense**：关联模型、可视化探索

#### 学习资源
- 书籍：《交互式数据可视化》、《Tableau实战》
- 课程：Plotly可视化、Tableau认证
- 文章：交互式图表设计、仪表板最佳实践
- 工具：Plotly、Tableau、Power BI

### 4. 机器学习基础

#### 监督学习
- **分类算法**：决策树、随机森林、支持向量机、逻辑回归
- **回归算法**：线性回归、岭回归、LASSO回归、集成方法
- **模型评估**：准确率、精确率、召回率、F1分数、ROC曲线
- **模型选择**：交叉验证、网格搜索、正则化

#### 无监督学习
- **聚类算法**：K-means、层次聚类、DBSCAN
- **降维技术**：PCA、t-SNE、LDA
- **关联规则**：Apriori算法、FP-Growth

#### 学习资源
- 书籍：《机器学习实战》、《统计学习方法》
- 课程：机器学习基础、Scikit-learn实战
- 文章：分类算法对比、聚类方法选择
- 工具：Scikit-learn、XGBoost、LightGBM

### 5. 领域应用

#### 商业分析
- **销售分析**：销售趋势、产品表现、区域分析
- **客户分析**：客户细分、客户生命周期价值、客户流失预测
- **市场分析**：市场趋势、竞争分析、定价策略
- **供应链分析**：库存优化、需求预测、物流优化

#### 金融分析
- **风险评估**：信用风险、市场风险、操作风险
- **投资分析**：投资组合优化、资产定价、市场预测
- **欺诈检测**：异常交易检测、模式识别
- **财务分析**：财务报表分析、财务预测

#### 科学分析
- **实验数据分析**：实验设计、结果分析、显著性检验
- **趋势分析**：时间序列分析、预测模型
- **数据挖掘**：模式发现、关联分析、异常检测

#### 学习资源
- 书籍：《商业数据分析》、《金融数据分析》
- 课程：商业智能、金融建模
- 文章：客户细分方法、风险评估模型

## 实践项目

### 1. 数据处理项目
- 导入和清洗大型数据集
- 执行特征工程
- 准备数据用于后续分析
- 评估数据质量和处理效果

### 2. 统计分析项目
- 执行描述性统计分析
- 设计和执行假设检验
- 进行回归分析
- 生成统计分析报告

### 3. 数据可视化项目
- 创建交互式仪表板
- 设计多种类型的可视化
- 呈现复杂数据关系
- 优化可视化效果

### 4. 机器学习项目
- 构建分类或回归模型
- 执行模型评估和选择
- 优化模型参数
- 解释模型结果

### 5. 领域应用项目
- 选择特定领域（如商业、金融、科学）
- 定义分析问题和目标
- 执行端到端分析
- 生成领域特定的分析报告

## 评估标准

- 能够清洗和处理复杂数据集
- 能够应用统计方法进行分析和假设检验
- 能够创建交互式数据可视化和仪表板
- 能够构建和评估机器学习模型
- 能够解决特定领域的分析问题

## 学习时间规划

- **第1-3周**：数据处理和特征工程
- **第4-6周**：统计分析方法
- **第7-9周**：高级数据可视化
- **第10-12周**：机器学习基础
- **第13-16周**：领域应用项目

## 下一步学习

完成中级阶段后，你可以进入高级阶段，学习高级统计方法、深度学习、大数据处理、分析工程和业务洞察等内容。
